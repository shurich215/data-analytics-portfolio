### [Ride Revenue Analysis](/ride-revenue-analysis)

Live Dashboard: https://public.tableau.com/app/profile/shu.richardson/viz/RideRevenueAnalysis/Summary

**Objective:**  
Analyze ride booking data to evaluate revenue performance, booking outcomes, and operational efficiency.

**What This Project Demonstrates:**

- Revenue analysis and performance tracking  
- KPI development (revenue, bookings, cancellation rate, completion rate)  
- Operational analysis (cancellations, incomplete rides, service reliability)

- Dashboard development in Tableau  

**Business Impact:**  
Identifies revenue drivers, highlights operational inefficiencies, and provides insights to improve service performance and customer experience.
